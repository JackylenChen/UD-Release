# 数据提供方 API 服务发布文档

``` 引言
数据提供方是数链平台上，向需求方提供数据的一种商家提供一组数据 API 服务。
它实现了一组以模型计算为核心的数据处理，多方计算混淆数据的业务过程，并且提供了一组 restful API 风格的简单接口。
```
