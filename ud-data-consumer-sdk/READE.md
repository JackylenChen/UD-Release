
# 数据输出服务 SDK 发布文档

``` 引言
数据输出服务 SDK 是数链平台提供的一个数据服务客户端，是为了帮助想接入数链平台的商家们提供一种快速简单的使用工具。
```

## 1.3.0-RELEASE （2018-08-24）

``` 发布说明
数链平台数据输出服务 SDK 可用版本发布，版本为 1.3.0。作为第一个可扩散版本，它提供了接入平台的基本支持：

1，支持接入数链平台
2，支持对 EOS 区块链的访问方式
3，支持一种基于 EOS 区块链合约的交易模型
4，支持一种本地管理交易订单的方式
5，...
```
